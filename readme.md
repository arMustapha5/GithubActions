## Test Selenium in Docker


#### Prerequisite

env
```
MAIL_SERVER=mailhog
MAIL_PORT=1025
MAIL_USERNAME=
NOTIFICATION_EMAIL= 
SLACK_WEBHOOK_URL=
GITHUB_REPOSITORY=

```


### Run

```
 ./run-docker-test.sh (linux)
 ./run-docker-test.bat (windows)
```
